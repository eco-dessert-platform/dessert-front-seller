# Redux 가이드

Redux를 사용한 상태 관리 실전 가이드입니다.

## 📚 가이드 목록

### [Reducer 생성하기](./create-reducer.md)

reduxMaker를 사용하여 새로운 reducer를 만드는 방법을 단계별로 안내합니다.

**주요 내용**:

- reduxMaker 기본 사용법
- 비동기 요청 정의하기
- 로컬 상태 및 리듀서 작성하기
- Store에 등록하기

**소요 시간**: 약 10분

### [비동기 작업 처리하기](./async-operations.md)

API 호출 및 복잡한 비동기 로직을 처리하는 방법을 배웁니다.

**주요 내용**:

- API 함수 작성하기
- Saga Effect 활용하기 (call, put, all, race)
- 에러 처리 및 재시도
- 병렬/순차 API 호출

**소요 시간**: 약 15분

---

### [성능 최적화](./performance.md)

Redux 애플리케이션의 성능을 최적화하는 방법을 배웁니다.

**주요 내용**:

- 메모리 관리 (initialize/initializeAll)
- useSelector 최적화
- Memoized Selector 사용
- 대용량 데이터 처리

**소요 시간**: 약 20분

### [Best Practices](./best-practices.md)

Redux 사용 시 권장사항과 안티패턴을 배웁니다.
