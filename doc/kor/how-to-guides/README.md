# How-to Guides (실용 가이드)

실제 개발에 필요한 구체적인 방법을 단계별로 안내합니다.

## 📑 가이드 목록

### Redux 가이드

Redux Toolkit과 Redux Saga를 사용한 상태 관리 실전 가이드입니다.

#### [Reducer 생성하기](./redux/create-reducer.md)

reduxMaker를 사용하여 새로운 reducer를 만드는 방법을 배웁니다.

**이런 분께 추천합니다**:

- 새로운 기능을 추가하려는 개발자
- Redux 구조가 처음인 개발자

**소요 시간**: 약 10분

#### [비동기 작업 처리하기](./redux/async-operations.md)

API 호출, 비동기 로직 처리 방법을 배웁니다.

**이런 분께 추천합니다**:

- API 통신이 필요한 기능을 개발하는 개발자
- 복잡한 비동기 플로우를 다루는 개발자

**소요 시간**: 약 15분

#### [성능 최적화](./redux/performance.md)

Redux 애플리케이션의 성능을 최적화하는 방법을 배웁니다.

**이런 분께 추천합니다**:

- 성능 문제를 겪고 있는 개발자
- 대용량 데이터를 다루는 개발자

**소요 시간**: 약 20분

#### [Best Practices](./redux/best-practices.md)

Redux 사용 시 권장사항과 안티패턴을 배웁니다.

**이런 분께 추천합니다**:

- 코드 품질을 향상시키고 싶은 개발자
- 팀 컨벤션을 정립하려는 개발자

**소요 시간**: 약 15분

### 스타일링 가이드

TailwindCSS와 shadcn/ui를 사용한 스타일링 가이드입니다.

#### [테마 커스터마이징](./styling/customize-theme.md)

프로젝트의 테마를 커스터마이징하는 방법을 배웁니다.

**이런 분께 추천합니다**:

- 브랜드 컬러를 적용하려는 개발자
- 다크/라이트 모드를 커스터마이징하려는 개발자

**소요 시간**: 약 10분

## 가이드 사용 방법

1. **목적에 맞는 가이드 선택**: 위의 설명을 참고하여 필요한 가이드를 선택하세요
2. **단계별 따라하기**: 각 가이드는 실습 위주로 구성되어 있습니다
3. **예제 코드 활용**: 제공된 예제 코드를 직접 수정해보세요
4. **문제 해결**: 막히는 부분이 있다면 [문제 해결](../resources/troubleshooting.md)을 참고하세요
