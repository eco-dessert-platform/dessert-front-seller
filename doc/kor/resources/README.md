## 기여하기

문서 개선에 기여하고 싶으신가요?

1. **문서 오류 발견**: GitHub Issue로 알려주세요
2. **새로운 가이드 제안**: Discussion에서 제안해주세요
3. **Pull Request**: 직접 문서를 개선해주세요

## 도움이 더 필요하신가요?

- **버그를 발견했다면**: [GitHub Issues](../../issues)에 등록해주세요
- **질문이 있다면**: [GitHub Discussions](../../discussions)를 이용해주세요
- **문서 개선 제안**: Pull Request를 보내주세요
