# 동적 라우팅 구조

- `src/pages/url/` 폴더 내에 `[param]` 형태로 동적 라우팅 구현
- 예시: `/url/[aid]/Sample.tsx` → `/url/123/Sample`
- **파일명 규칙**: feature와 구분을 위해 페이지 컴포넌트는 `Page` 접미사 사용 권장 (필수는 아님)

## 주의사항

- `src/pages/url/` 폴더에는 **공통으로 사용되는 페이지만** 추가합니다
- `router.tsx`에서 `src/pages/url/**/*.tsx` 경로의 모든 파일을 자동으로 라우트로 등록합니다
- 특정 도메인이나 기능에 종속된 페이지는 이 폴더에 추가하지 않습니다
