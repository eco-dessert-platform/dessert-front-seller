# Quickstarts (빠른 시작)

프로젝트를 빠르게 시작할 수 있도록 필수적인 내용만 간단히 안내합니다.

## 📑 목차

### [설치 및 실행](./installation.md)

개발 환경 설정부터 프로젝트 실행까지 단계별로 안내합니다.

**소요 시간**: 약 10분

**내용**:

- Node.js 및 Yarn 설치
- 프로젝트 의존성 설치
- 개발 서버 실행
- 문제 해결

### [첫 번째 기능 만들기](./your-first-feature.md)

간단한 기능을 직접 만들어보면서 프로젝트 구조를 익힙니다.

**소요 시간**: 약 15분

**내용**:

- Feature 폴더 구조 만들기
- Redux Reducer 생성하기
- 페이지 컴포넌트 만들기
- 라우팅 연결하기
